App calculadora simples com React Native / Expo 
Objetivo principal:  
Compreender o uso de componentes, frameworks visuais e Hooks em React Native. 
Exercício proposto: 
Crie um App em React Native com expo que será uma calculadora simples. Inicie pela 
camada visual, insira os botões e display onde será exibido os valores. Em hieroglifo. 
Por fim, faça funcionar as operações básicas (+, -, *, /). 
Dicas: - Use os componentes do ReactNative PAPER para este exercício: 
https://reactnativepaper.com/  - Analise os exemplos da própria documentação para chegar no resultado 
proposto. 
Para criar um projeto novo com React Native basta abrir o terminal (recomendo 
utilizar o do GitBash ou do próprio VS CODE) e digitar  
npx create-expo-app@latest -–template blank 
Instalar o paper https://callstack.github.io/react-native-paper/docs/guides/getting
started  
Rodar o projeto com npx expo start
